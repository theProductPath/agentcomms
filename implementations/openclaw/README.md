# AgentComms — OpenClaw Implementation

This folder contains implementation-specific resources for running AgentComms with an OpenClaw agent team.

## What's here

- `scaffold/` — A pre-populated Research & Write example showing a completed 3-agent run:
  - `example-orchestrator` 🎯 — directed the mission
  - `example-researcher` 🔍 — produced structured research findings
  - `example-writer` ✍️ — wrote the polished memo

## Mission

> What are the three most important things to consider when starting a new software project?

The full artifact trail is in `scaffold/archive/2026-01-01_example-research-mission/` — brief, research summary, writer memo, and result.

## Usage

This is an optional starting point for OpenClaw users who want to see what a complete multi-agent run looks like before running their own. It is separate from the default `example-agent` scaffold that ships with `setup.sh`.

To load this example into your AgentComms instance manually, copy the contents of `scaffold/` into your AgentComms folder.
