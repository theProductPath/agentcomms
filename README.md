# AgentComms — site companion

The marketing and landing pages for [AgentComms](../agentcomms/README.md). This folder holds the website source, assets, and deployment configuration for `agentcomms.theproductpath.com`.

For the product itself (concepts, spec, implementation), see [`../agentcomms/`](../agentcomms/README.md).

## GitHub Pages

This repo publishes the static site from the repository root with GitHub Pages. The deployment workflow is in `.github/workflows/pages.yml`, and the custom domain is preserved in `CNAME`.

The Pages source for this repository should be set to **GitHub Actions** in the repository settings.
